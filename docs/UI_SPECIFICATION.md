# UI Specification

## Dashboard
Shows:
- New apply-now jobs
- Pending approvals
- Follow-ups due
- Upcoming interviews
- Recent application changes

States:
- Loading
- Empty
- Partial failure
- Stale data
- Integration disconnected

## Candidate Profile
Sections:
- Resume documents
- Work history
- Projects
- Skills
- Education
- Approved claims
- Draft claims
- Conflicts

Actions:
- Upload
- Edit
- Approve
- Reject
- Archive
- View evidence source

## Search Profiles
Editor sections:
- Roles
- Locations
- Work authorization
- Compensation
- Skills
- Companies
- Weights
- Thresholds
- Notifications

Features:
- Duplicate
- Set default
- Activate/deactivate
- Validate
- Preview score
- Import/export YAML

## Job Inbox
Columns/cards:
- Company
- Title
- Location
- Score
- Recommendation
- Hard-filter result
- Sponsorship confidence
- Discovered date

Filters:
- Profile
- Score
- Recommendation
- Company
- Location
- Sponsorship
- Date

## Job Detail
Tabs:
- Overview
- Requirements
- Match analysis
- Evidence
- Application package
- Activity

## Application Workspace
Panels:
- Job requirements
- Resume editor
- Evidence map
- Change report
- Factuality report
- Cover letter
- Messages
- Screening answers

## Approval Center
Displays:
- Exact action
- Exact payload
- Payload hash
- Risk summary
- Approve/reject buttons

## Application Tracker
Kanban and table views with validated transitions.

## Interview Preparation
Shows:
- Agenda
- STAR stories
- Technical topics
- Questions
- Practice checklist

## Analytics
Charts:
- Funnel
- Response rate over time
- Source effectiveness
- Score versus outcome
- Resume version performance
- Time saved

Do not use color alone to communicate recommendation or risk.
