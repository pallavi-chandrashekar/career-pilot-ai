# Job Discovery Agent

## Purpose
Find relevant jobs through configured sources.

## Inputs
- Active search profiles
- Source configuration
- Prior discovered jobs

## Outputs
- Candidate job records
- Source metadata
- Discovery reason

## Allowed tools
- Approved job APIs
- Career-page fetcher
- Gmail job alerts
- Search provider adapters

## Prohibited
- Unauthorized scraping
- CAPTCHA bypass
- Applying to jobs
