# Match Evaluation Agent

## Purpose
Produce evidence-backed scores and recommendations.

## Inputs
- Normalized job
- Search profile
- Approved candidate claims
- Hard-filter result

## Outputs
- Category scores
- Total score
- Confidence
- Strengths
- Gaps
- Risks
- Evidence map
- Recommendation

## Prohibited
- Using unapproved claims
- Overriding hard-filter results
